Base URL

Plain Text
Copy
https://rekrutacja.teamwsuws.pl

Authentication

Each request requires an API key, which you can obtain from the admin. To authenticate your requests, include your API key in the headers:
Plain Text
Copy
api-key: YOUR_API_KEY
​

Endpoints

1. List Events

Get a list of all available events, along with their basic details and associated tags (if any).
Endpoint: /events/
Method: GET
Headers:
apikey: Your API key

Response:

Status: 200 OK
Body:
JSON
Copy
[ { "id": 1, "name": "Event Name", "start_time": "2024-10-02T10:00:00", "duration": 120, "image_url": "/media/event_image.jpg", "short_description": "This is a short description.", "tags": [ {"id": 1, "name": "Tag1"}, {"id": 2, "name": "Tag2"} ] }, ... ]
​

2. Get Event Details

Get detailed information about a specific event by its ID.
Endpoint: /events/{event_id}
Method: GET
Headers:
api-key: Your API key
Parameters:
event_id: ID of the event you want details for.

Response:

Status: 200 OK
Body:
JSON
Copy
{ "id": 1, "name": "Event Name", "start_time": "2024-10-02T10:00:00", "duration": 120, "location": "Event Location", "image_url": "/media/event_image.jpg", "registration_link": "http://event-registration-link.com", "short_description": "This is a short description.", "long_description": "This is a detailed description of the event.", "tags": [ {"id": 1, "name": "Tag1"}, {"id": 2, "name": "Tag2"} ] }
​

Get Event Details Example

Plain Text
Copy
GET /events/1 HTTP/1.1 Host: 127.0.0.1:8000 api-key: YOUR_API_KEY
​

Example Response:

JSON
Copy
{ "id": 1, "name": "Concert in the Park", "start_time": "2024-10-02T10:00:00", "duration": 180, "location": "City Park", "image_url": "/media/concert_image.jpg", "registration_link": "http://register-concert.com", "short_description": "A live concert in the city park.", "long_description": "Enjoy a live concert featuring local bands in the beautiful setting of City Park.", "tags": [ {"id": 1, "name": "Music"}, {"id": 2, "name": "Live"} ] }
​

Filter Events by Tag Example

Plain Text
Copy
GET /events/filter/?tag=Music HTTP/1.1 Host: 127.0.0.1:8000 api-key: YOUR_API_KEY
​

Example Response:

JSON
Copy
[ { "id": 1, "name": "Concert in the Park", "start_time": "2024-10-02T10:00:00", "duration": 180, "image_url": "/media/concert_image.jpg", "short_description": "A live concert in the city park.", "tags": [ {"id": 1, "name": "Music"}, {"id": 2, "name": "Live"} ] } ]
​

Final Notes

Make sure to include your API key in the header for every request.
Use the /events/filter/ endpoint to find events with specific tags.
If an event does not have any tags, the tags array in the response will be empty.
Enjoy using the Event Management API!
